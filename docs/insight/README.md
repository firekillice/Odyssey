## 算法的理解
#### 排序
* 对某个数组进行排序，其实是在其他的大循环中进行的，是"楚门的世界"中；同理，某个大数组也可以划分为更小的数组进行排序，所以可以假想有个巨大的数组
* 算法只是一种思路，可以用多种方式进行实现
* 找到算法的某种趋近状态很重要
* 整体思路：划分划分再换分
* 写代码的时候，先确定大循环，然后一层层分析嵌套来思考
